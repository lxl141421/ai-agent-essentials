# 前端工程师

> 你的 Agent 需要能看懂网页。

---

## 你的需求

- 能操作浏览器（点击、截图、填表）
- 能分析网页结构
- 能跑 npm / vite 等工具链
- 能帮你写组件、调样式

## 推荐配置

### 必装

```bash
pip install browser-use
```

- **browser-use** - AI 驱动的浏览器自动化
- **终端 + 文件工具** - 跑 npm、改代码

### 加装

- **MCP 生态** - Playwright MCP、Browserbase MCP
- **Prompt 管理** - 组件生成模板放 Git

### 预算

- 月费: $10-30

## 典型工作流

```
1. "帮我看看这个页面的布局" -> browser-use 截图分析
2. "写一个登录组件" -> 终端跑 npm + 生成代码
3. "这个按钮样式不对" -> browser-use 截图 + 修改 CSS
4. "对比竞品首页" -> browser-use 打开多个页面
```
