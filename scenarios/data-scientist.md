# 数据科学家

> 你的 Agent 需要能处理数据和文档。

---

## 你的需求

- 能查阅论文和技术文档
- 能运行 Python 数据分析脚本
- 能管理 Jupyter notebook
- 能记住你的研究上下文

## 推荐配置

### 必装

```bash
pip install llama-index mem0
```

- **LlamaIndex** - 索引论文、文档、数据集
- **mem0** - 记住研究上下文和偏好
- **终端 + 文件工具** - 跑 Python、管理 notebook

### 加装

- **Token 监控** - 论文检索消耗大，需要监控

### 预算

- 月费: $30-80（RAG 检索消耗较多 token）

## 典型工作流

```
1. "这篇论文的方法是什么" -> LlamaIndex 检索论文
2. "帮我跑一下这个数据集" -> 终端执行 Python
3. "上次实验的结论是什么" -> mem0 检索记忆
4. "写个数据清洗脚本" -> 读数据结构 + 生成代码
```
