# 评估 / 可观测性

> 看不见就管不了。

---

## 为什么需要？

- Agent 决策过程是黑盒
- Token 消耗需要追踪
- 工具调用需要审计
- 质量需要评估

## 推荐方案

### LangSmith (LangChain 官方, 有免费额度)

Agent 执行链路追踪, Token 消耗统计, 评估框架, Prompt 版本管理。适合 LangChain 生态。

### Weights & Wies (有免费额度)

ML 实验追踪, Agent 行为可视化, 模型性能对比。适合有 ML 需求的团队。

### Phoenix (~8k Star, Apache 2.0, 开源)

开源免费, LLM 专用可观测性, RAG 检索质量分析, 可本地部署。不想用云服务的首选。

## 最低限度

不想引入额外工具的话，至少做到:
- 所有工具调用和结果留日志
- 每次对话记录 token 消耗
- 工具调用失败有告警
- 每天/每周统计 API 费用
